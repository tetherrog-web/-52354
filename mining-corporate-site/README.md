# 矿业集团企业网站

## 项目概述

这是一个完整的矿业集团企业网站项目，包含20个专业HTML页面、现代CSS样式、交互式JavaScript脚本和第三方库集成（Bootstrap + ECharts）。

## 项目结构

```
mining-corporate-site/
├── index.html              # 首页
├── about.html              # 关于我们
├── history.html            # 发展历程
├── culture.html            # 企业文化
├── projects.html           # 投资项目
├── project-detail.html     # 项目详情
├── global-operations.html  # 全球运营
├── mines.html              # 全球矿山
├── mine-detail.html        # 矿山详情
├── news.html               # 新闻中心
├── news-detail.html        # 新闻详情
├── investors.html          # 投资者关系
├── reports.html            # 财务报告
├── stock.html              # 股票信息
├── sustainability.html     # 可持续发展
├── environment.html        # 环保责任
├── safety.html             # 安全管理
├── careers.html            # 招聘信息
├── contacts.html           # 联系我们
├── css/
│   └── style.css           # 全局样式表
├── js/
│   └── main.js             # 主要JavaScript脚本
└── assets/
    ├── images/             # 图片资源
    └── data/               # JSON数据文件
```

## 快速开始

### 1. 解压项目
```bash
unzip mining-corporate-site.zip
cd mining-corporate-site
```

### 2. 启动开发服务器
使用Python内置服务器：
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

或使用Node.js的http-server：
```bash
npx http-server
```

### 3. 访问网站
在浏览器中打开：
```
http://localhost:8000
```

## 自定义配置

### 修改颜色主题
编辑 `css/style.css` 的CSS变量：
```css
:root {
  --primary-color: #0b2a4a;      /* 深蓝色 */
  --secondary-color: #1a3a5a;    /* 次蓝色 */
  --accent-color: #d4af37;       /* 金色强调 */
  /* ... 其他变量 */
}
```

### 更新公司信息
在各页面的HTML中搜索并替换：
- 公司名称：`紫金矿业集团` → 你的公司名
